# DeepSzpont - A study investigating the human ability to distinguish between authentic and AI-generated videos.

## The project consists of three main components developed to facilitate the study:

* A curated video dataset: A collection of 20 video samples, comprising 7 authentic clips sourced from Pixabay and 13 AI-generated sequences.

* A Python-based preprocessing tool: A custom script designed to crop footage and remove watermarks to ensure visual consistency for the experiment.

* A web assessment platform: A dedicated web application allowing study participants to view and evaluate the video samples

### Locally set up models: (I2V oraz T2V)
